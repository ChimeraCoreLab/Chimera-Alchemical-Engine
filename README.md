**CHIMERA CORE: ALCHEMICAL ENGINE** is a hybrid cinematic and interactive psychological horror environment built entirely within **Godot 4.6.2**. This project serves as a digital monument, synthesizing three years of unstructured relational logs, hardware constraints, and personal artifacts into a computable narrative.

## I. System Architecture
This project achieves **Technical Sovereignty** by bypassing traditional video editing software. All logic, from the frame-accurate subtitle typewriter system to the recursive visual glitches, is hard-coded into the engine via `GDScript`.

### Key Features:
*   **Neural Sequencer**: A specialized automated system that synchronizes on-screen text and visual triggers with atomic `.wav` audio nodes.
*   **Procedural VHS/CRT Shaders**: Real-time GLSL post-processing simulating "Digital Rot" and analog signal degradation.
*   **Atmospheric Spatialization**: Integrated 3D audio triggers and cinematic camera switching via `Area3D` nodes for non-linear storytelling.
*   **Multi-modal Content**: Real-time rendering of high-resolution hand-drawn artifacts (Nodes 33-40) and 3D environment synthesis.

## II. Execution Logic
1.  **Clone**: `git clone https://github.com/ChimeraCoreLab/Chimera-Alchemical-Engine.git`
2.  **Environment**: Open the project using **Godot 4.6.2 Stable** (Optimized for Mobile/Compatibility).
3.  **Uplink**: Load `res://scenes/scene_1_genesis.tscn`.
4.  **Synthesis**: Press `F5` to initiate the cinematic sequence.

## III. The Sovereign Artifacts
The `res://Artifacts/` directory contains over 100 visual data nodes distilled from digital archaeology.
*   **Origin**: All assets were meticulously recreated by hand or captured via system snapshots by **USR_00**.
*   **Styles**: Ranging from low-fidelity 8-bit artifacts to high-resolution 32-bit transparent PNGs.
*   **Integrity**: 100% original signal purity, designed to evoke psychological dread and "Uncanny Valley" responses.
