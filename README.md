## I. System Architecture
This project achieves **Technical Sovereignty** by bypassing traditional video editing software. All logic, from the frame-accurate subtitle typewriter system to the recursive visual glitches, is hard-coded into the engine via `GDScript`.

### Key Features:
*   **Neural Sequencer**: An automated, data-driven system that synchronizes subtitles and visual triggers with atomic `.wav` audio nodes.
*   **Procedural VHS/CRT Shaders**: Real-time GLSL post-processing simulating "Digital Rot" and analog degradation.
*   **Atmospheric Spatialization**: Integrated 3D audio triggers and cinematic camera switching via `Area3D` nodes.
*   **Multi-modal Content**: Real-time rendering of high-resolution hand-drawn artifacts and 3D environment synthesis.

## II. Execution Logic
1.  **Clone**: `git clone https://github.com/ChimeraCoreLab/Chimera-Alchemical-Engine.git`
2.  **Environment**: Open the project using **Godot 4.6.2 Stable**.
3.  **Uplink**: Load `res://scenes/scene_1_genesis.tscn`.
4.  **Synthesis**: Press `F5` to initiate the cinematic rendering.
